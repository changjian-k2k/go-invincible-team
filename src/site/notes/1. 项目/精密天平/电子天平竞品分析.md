# 赛多利斯 Sartorius

实验室天平产品类别

超微量实验室天平：最大量程为2.1g，分度值（可读性）为0.1ug。
大量程微量天平：最大量程从32g到111g，可读性为1ug或2ug，是在大容器中称重极小样品量的理想型号。
半微量实验室天平：半微量实验室天平的可读性为0.01mg或10ug，最大量程高达220g。
分析实验室天平：分析实验室天平是实验室中使用最多的天平。其最大量程介于60g至520g之间，**可读性为0.1mg**，通常用于称量小样品。
精密实验室天平和秤：大量程天平的**可读性介于1mg和1g之间**，最大量程高达14.2kg。

分析天平比精密天平更加精确。分析天平的可读性为0.1mg或0.0001g，而精密天平的可读性通常为≥1mg或≥0.001g。因此，分析天平的可读性至少是精密天平的10倍。

目前官网主推的天平为：Cubis II新一代半微量天平，全新设计的Quintix Pro。

# 梅特勒

## 梅特勒各系列定位

 梅特勒天平系列较多，对应的各系列的定位和使用场景如下：
 1. **MX 旗舰系列**：
    - **场景**：这是梅特勒的高端系列天平，适用于对称量精度、智能化操作和数据管理有极高要求的场景。比如顶级科研机构的前沿科学研究实验室，在进行复杂的实验项目且对数据的准确性和可追溯性要求极其严格时，MX 系列天平能够提供高度精准的称量结果和智能化的操作体验，满足科研人员对实验数据的高标准要求；大型制药企业的研发和质量控制实验室，在进行新药研发、药品成分分析等关键环节，需要高精度、可靠的称量设备来确保实验结果的准确性和一致性，MX 系列天平可以很好地满足这些需求。
2. **MR 主力系列**：
    - **场景**：作为主力系列，具有多功能性，能够适应各种不同的环境。适用于中等规模的实验室，如高校的专业实验室、中型药企的质量检测实验室等。在这些场景中，实验项目较为多样，需要天平具备多种称量功能和较好的环境适应性，MR 系列天平可以满足日常实验的各种称量需求，并且能够在不同的实验室环境条件下稳定工作。同时，对于一些需要经常进行现场称量的应用场景，如食品加工企业的生产现场、环境监测机构的户外采样现场等，MR 系列天平的便携性和稳定性也使其能够胜任。
3. **MA 实用系列**：
    - **场景**：该系列天平坚固耐用，主要用于简化日常工作。适合一般性的实验室工作场景，如企业的常规检测实验室、学校的教学实验室等。在这些场景中，天平的使用频率较高，但对称量精度的要求不像高端科研实验室那样苛刻，MA 系列天平的坚固耐用性能够保证其在长期使用过程中的稳定性和可靠性，减少设备的维护成本和更换频率。此外，对于一些对天平操作简便性有要求的用户，MA 系列的设计更加注重用户友好性，方便操作人员快速上手，提高工作效率。
4. **LA 基础系列**：
    - **场景**：这是一款入门级的天平系列，简单可靠，适合对天平功能要求相对简单的场景。例如，小型实验室、小型企业的质量检验部门等，在进行一些基本的称量操作时，LA 系列天平可以提供准确的称量结果，满足日常工作的需求。同时，对于教学用途，特别是在初级实验教学中，LA 系列天平的简单操作和基本功能可以帮助学生快速掌握天平的使用方法，是一种经济实惠的选择。
5. **XPR 系列**：
    - **分析天平产品线**：可读性高，具有卓越的称量性能和可靠的质量保证。适用于对称量精度要求较高的分析化学实验、材料科学研究等场景。例如，在化学实验室中进行样品的定量分析、纯度检测等实验时，XPR 分析天平能够准确地测量样品的质量，为实验结果的准确性提供保障；在材料科学研究中，对材料的成分分析、性能测试等环节，也需要高精度的称量设备来获取准确的实验数据2。
    - **微量天平产品线**：能够精确称量极微小的样品，可读性可达 6 位。常用于对微量物质进行称量的场景，如生命科学领域的细胞培养、蛋白质分析，以及环境科学领域的微量污染物检测等。在这些实验中，样品的量非常少，需要高精度的微量天平来准确测量，XPR 微量天平的高分辨率和高精度能够满足这些特殊的称量需求2。
6. **MS 系列**：
    - **场景**：属于精密天平系列，具有抗冲击、抗过载性能，确保称量结果的准确性。适用于工业生产中的质量控制环节，如电子制造行业中对电子元器件的称重、汽车制造行业中对零部件的称重等。在这些场景中，天平需要具备较高的稳定性和抗干扰能力，以保证在工业生产环境下能够准确地称量各种物品。此外，MS 系列天平内置多种称量应用程序，如计件称量、百分比称量、检重称量等，也适用于需要进行快速、批量称量的工业场景。
7. **XS 系列**：
    - **场景**：是分析天平系列中的高性能产品，其采用的革新性的网格秤盘能够将室内空气气流干扰降至最低，实现快速、稳定的称量结果。适用于对称量精度和稳定性要求极高的实验室环境，如高端质检机构的检测实验室、大型药企的研发实验室等。在进行高精度的分析实验、配方研发等工作时，XS 系列天平能够提供准确、可靠的称量数据，为实验的成功提供保障。

# 舜宇恒平
## 恒平各系列定位
1. **FA 系列分析天平**：
    - **场景**：这是舜宇恒平较为经典的分析天平系列，适用于对称量精度要求高的实验室场景，如高校、科研机构的化学实验室、生物实验室等。在进行药物研发、化学分析、材料研究等实验时，需要精确地称量各种化学试剂、药品、样品等，FA 系列天平的高精度（万分之一克及以上精度）能够满足这些实验的要求，确保实验数据的准确性。例如在高校的化学实验教学中，学生进行定量分析实验时，FA 系列天平可以为他们提供准确的称量结果。
2. **JA 系列电子精密天平**：
    - **场景**：该系列天平具有较高的精度和稳定性，适用于对精度要求较高的一般性称量场景。比如企业的质量检测部门、食品加工企业的实验室等。在质量检测过程中，需要对原材料、成品进行准确称量，以确保产品质量符合标准；食品加工企业在对食品原料进行称量时，JA 系列天平的稳定性和精度可以保证称量结果的可靠性。此系列天平还具备多种功能，如过载保护、单位转换等，方便用户使用5。
3. **MP 系列大称量天平**：
    - **场景**：主要用于大称量需求的场景，如工厂的仓库、物流中心等。在这些场所，需要对大量的货物进行称重，MP 系列天平的大称量范围（例如有 60kg、100kg、200kg 等规格）可以满足这种需求。比如在物流中心，对大件包裹、货物进行称重时，MP 系列天平能够快速、准确地完成称量工作，提高物流效率19。
4. **AE 系列微量分析天平**：
    - **场景**：适用于对微量物质进行精确称量的场景，如科研机构的微量分析实验、制药企业的药物研发等。该系列天平的精度非常高，能够准确称量极微量的样品，对于需要精确控制微量物质用量的实验和研究非常重要。例如在生物制药领域，对微量的药物成分进行称量和分析时，AE 系列微量分析天平可以提供准确的称量结果8。
5. **JY 系列电子天平**：
    - **场景**：这是一款适用于基础称量需求的天平系列，常用于学校的普通实验室、小型企业的实验室等。在这些场景中，主要进行一些常规的称量操作，对天平的精度要求相对不是特别高，JY 系列天平的性价比高，能够满足基本的称量需求，且操作简单方便，适合初学者和日常使用。
6. **电子密度天平系列**（如 MP3002J、JA3003J 等型号）：
    - **场景**：专门用于测量物体的密度，适用于对密度测量有要求的场景，如材料科学研究、珠宝鉴定、油品检测等行业。在材料科学研究中，需要准确测量材料的密度以分析其物理性质；珠宝鉴定中，通过测量宝石的密度可以辅助判断宝石的真伪和品质；油品检测时，密度是油品的重要参数之一，电子密度天平可以快速准确地测量油品的密度。
